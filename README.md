# ☕ Predictive Machine Learning Model for Annual Coffee Yield Forecasting  
### Case Study: Western Hararghe Zone, Ethiopia  

---

## 📌 Overview  

This project presents a predictive machine learning approach for forecasting annual coffee yield in the Western Hararghe Zone of Ethiopia. Coffee is a key agricultural commodity in the region, and accurate yield prediction is essential for improving farm management, policy planning, and resource allocation.

The model leverages historical agricultural data and machine learning techniques to estimate coffee yield based on environmental, soil, and farming-related factors.

---

## 🎯 Objectives  

- Develop a machine learning model for coffee yield prediction  
- Identify key factors influencing coffee production  
- Improve forecasting accuracy for agricultural planning  
- Support data-driven decision-making in the coffee sector  

---

## 📊 Dataset Description  

The dataset used in this study includes agricultural and environmental variables such as:

- Rainfall  
- Temperature  
- Soil properties  
- Fertilizer usage  
- Farming practices  
- Historical coffee yield (target variable)

> ⚠️ Note: The dataset may require preprocessing before modeling (handling missing values, encoding categorical variables, and normalization).

---

## ⚙️ Methodology  

The project follows a standard machine learning pipeline:

1. **Data Collection**
2. **Data Cleaning & Preprocessing**
   - Handling missing values  
   - Encoding categorical variables  
   - Outlier detection and treatment  
3. **Exploratory Data Analysis (EDA)**
   - Correlation analysis  
   - Visualization of feature distributions  
4. **Feature Engineering**
   - Selection of relevant predictors  
5. **Model Development**
   - Regression-based machine learning models  
   - Training and validation  
6. **Model Evaluation**
   - Mean Absolute Error (MAE)  
   - Root Mean Squared Error (RMSE)  
   - R² Score  
7. **Visualization of Results**
